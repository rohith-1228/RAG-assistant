# RAG AI Assistant

Milestone 1: Semantic Search
Milestone 2: PDF Search
Milestone 3: Streamlit UI
Milestone 4: Full RAG with LLM

## Setup

1. Clone the repo
2. Create a `.env` file in `milestone4-rag-assistant/` based on `.env.example`
3. Add your own `GOOGLE_API_KEY` from [Google Cloud Console](https://console.cloud.google.com/apis/credentials)
4. Install dependencies and run
